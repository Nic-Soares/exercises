Ela ento quer cobrir o treinamento de 90% dos funcionrios. Sabendo a quantidade de horas de treinamento tem um desvio padro de 2 horas e a empresa tem 500 funcionrios, quantas horas de treinamento ela buscar negociar para o ano?

# Definindo os parmetros
media <- 9  # Mdia de horas de treinamento por ano
desvio_padrao <- 2  # Desvio padro de horas de treinamento por ano

Exerccio 9

Faa um grfico da distribuio Chi-quadrado com diferentes graus de liberdade. Para que valores de graus de liberdade entre 10, 20 e 30 a distribuio apresenta valores mais dispersos?

library(latex2exp)

# Definindo os graus de liberdade
df_values <- c(10, 20, 30)

# Criando o layout dos grficos
par(mfrow = c(1, 3))

# Loop para criar os grficos para diferentes graus de liberdade
for (df in df_values) {
  # Gerando valores de x
  x <- seq(0, 60, 1)
  
  # Calculando a densidade de probabilidade
  prob <- dchisq(x, df = df)
  
  # Plotando a densidade de probabilidade
  plot(x,
       prob,
       type = 'l',
       xlab = TeX('Valor'),
       ylab = TeX('Funo Densidade de Probabilidade'),
       col = 'darkblue',
       main = TeX(sprintf('PDF, Chi-quadrado(df=%d)', df)))
  
  # Calculando a probabilidade acumulada
  prob_acumulada <- pchisq(x, df = df)
  
  # Plotando a probabilidade acumulada
  plot(x,
       prob_acumulada,
       type = 'l',
       xlab = TeX('Valor'),
       ylab = TeX('Funo Probabilidade Acumulada'),
       col = 'darkblue',
       main = TeX(sprintf('CDF, Chi-quadrado(df=%d)', df)))
}

# Restaurando o layout padro
par(mfrow = c(1, 1))
